Commands available from the three vertical dots menu

Alerts alarm_add Configure various actions that will take place when an unusually low or high heart rate is received. See here for more.
Alarm off
Hide/show Toggles hiding or showing individual patients.
Charts on/off Toggles showing the charts.
Settings Configure various settings that are required for Cyfer Health Monitor to function. See here for more.
Log A debugging command. This shouldn't be necessary to use.
Simulate on/off Just used to demo Cyfer Health. It creates artificial patients and artificial heart rate readings.
Open folder Open the folder/directory where the readings files are saved. The folder icon turns green when a new requested file has arrived.
Help help This help page.
About info
Exit Terminates Cyfer Health Monitor. The patients will no longer be monitored unless another Monitor is running.
